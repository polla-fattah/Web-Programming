# Backend Web Programming (PHP + Laravel)

A comprehensive course teaching backend web development with PHP and Laravel framework, from fundamentals to production deployment.

## 📚 Course Overview

**Target Audience:** Students with foundational knowledge of SQL and Object-Oriented Programming  
**Duration:** 14 weeks  
**Focus:** Building real, production-ready backend applications

## Learning Outcomes

By the end of this course, you will be able to:

- Build Laravel applications using MVC architecture and Eloquent ORM
- Design relational database schemas and write migrations
- Implement proper validation and authorization in your applications
- Create RESTful APIs with API resources and rate limiting
- Write feature tests for critical application flows
- Deploy applications to production environments

## Required Tools

Install these before Week 1:

- **PHP 8.x** - Latest PHP version
- **Composer** - PHP package manager
- **Laravel** - Installed via Composer
- **Database** - MySQL or PostgreSQL
- **Git & GitHub/GitLab** - Version control
- **VS Code** - Code editor
- **Postman or Insomnia** - API testing tool

## Core Methodology: The Course Rule

Every feature developed in this course follows this structure:

1. **Migration** - Create/modify database tables with constraints
2. **Model & Relationships** - Define data structure and relationships
3. **Validation** - Validate user input with request classes
4. **Authorization** - Control access with policies or gates
5. **Output** - Return data via Blade templates or APIs
6. **Testing** - Write feature tests for critical flows (starting Week 12)

## Getting Started with the Slides

### Installation

```bash

# Start the development server
pnpm dev
```

## Course Rules Summary

Every feature must include:
1. Migration / DB constraint
2. Model + relationship (if needed)
3. Request validation
4. Authorization (policy/gate)
5. Output (Blade or API)
6. At least 1 feature test (from Week 12 onwards)
